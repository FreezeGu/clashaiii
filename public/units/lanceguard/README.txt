Lanceguard (Lancia) sprite animation
====================================

Put your PNG files in these folders (same number format: 1.png, 2.png, …):

  idle/   → 7 frames:  1.png … 7.png  (deck preview + in battle when standing still)
  walk/   → 7 frames:  1.png … 7.png  (in battle when moving)
  attack/ → 5 frames:  1.png … 5.png  (sword overlay when attacking)

Where they show up:
  - Deck/collection cards: idle animation
  - Main battle: idle when still, walk when moving, attack overlay when attacking
  - Card inspection mini battle: same as main battle

If files are missing, the game falls back to colored circles.
