Frost Mage (maga) sprite animation
=================================

Same structure as Lanceguard. Put PNGs in (number format 1.png, 2.png, …):

  idle/   → 8 frames:  1.png … 8.png
  walk/   → 7 frames:  1.png … 7.png
  attack/ → 5 frames:  1.png … 5.png

Used in deck preview (idle) and in battle (idle when still, walk when moving, attack when casting).
